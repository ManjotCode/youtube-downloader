# Youtube-downloader
<br>
This is python program for download youtube videos
<br>
 <h2>Requirment</h2>
install pytube

```pip install pytube```
<br>
if pip not working activate python envirment
<br>
```python3 -m venv yt```
<br>
```source yt/bin/activate```
<h2>Run</h2>  
To run this program use
<br>

```python youtube_downloads.py ```
<br>
## Exit
To stop use 

```C-c```
